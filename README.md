# my-desk

I'm making a personal website! It's my desk, digitalized, with interactable objects used as navigation.

**Live:** https://my-desk-three.vercel.app

## How it's built

I'm using Next.js 16, TypeScript, Tailwind v4, and Motion to build out one interactive scene, with everything else on the website static. For now, I'm using placeholders, but I eventually want to hand-draw all the objects in my website!

## How it's _being_ built

This is a project for learning! I'm using Claude as a personal tutor throughout the process, but every line of code is going to be written by me.
